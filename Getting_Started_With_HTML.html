Xinya's Study Notes about HTML

General Concepts

Web
user - browser - internet -(HTTP Request)- server

HTML Element
<start_tag> Content <end_tag>
An inline element works in the existing line while a block element creat a new block to make change in it

Tags and Attributes
HTML elements are defined by tags
Attibutes can provide more information about the element, usually specified in the start tag in the form of name/value pairs

Useful Tags

bold tag
- to print text in bold
<b> content </b>

italicize tag
- to print text in italics
<em> content </em>

anchor tag
- to make a link
<a href="url"> link </a>

image tag
- to add an image inline (void tag)
<img src="url" alt="if img not loading display this">

break tag
- to start a new line inline (void tag)
first line <br> second line

paragraph tag
- to define a paragraph, with margin added before and after
<p> paragraph </p>

span tag
- to define an inline section to format
<span> inline section </span>

division tag
- to define a section or grouped section of block elements to format
<div> block elements </div>

HTML Document Format

<!DOCTYPE html>
<html>

<head> -to include metadata, javascript, CSS...
<title> My Title </title>
</head>
<body> -to display contents
<h1> My Headlines </h1>
<p> My Paragraphs </p>
</body>
</html>
